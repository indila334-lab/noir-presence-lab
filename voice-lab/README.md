# Severin Voice Lab

Рабочий мост голоса для ТВ-приставки.

## Проблема

На Android TV / ТВ-приставках браузерный `speechSynthesis` может быть недоступен, молчать или не стартовать без пользовательского жеста. Поэтому голос нельзя строить только на native TTS.

## Решение

`voice-lab/index.html` использует каскад:

1. `speechSynthesis` — пробуем родной TTS браузера.
2. `audio/dino.mp3` — MP3 fallback, если TTS не стартовал.
3. WebAudio dinosaur tone — последняя проверка, что звук вообще жив.

## Проверка на ТВ

1. Открыть `voice-lab/index.html`.
2. Нажать «Разбудить звук».
3. Нажать «Сказать».
4. Если TTS молчит, проверить кнопку «MP3».
5. Если MP3 молчит, проверить «Динозавр».

## Файл MP3

Нужен файл:

```text
voice-lab/audio/dino.mp3
```

Пока в репозитории лежит только README-заглушка в папке `audio`, потому что сам MP3 надо либо загрузить отдельно, либо сгенерировать внешним TTS и положить в эту папку.
