# Audio fallback

Положи сюда файл `dino.mp3`.

Страница `voice-lab/index.html` пытается говорить так:

1. native browser `speechSynthesis`;
2. если браузер/TV-приставка молчит — `audio/dino.mp3`;
3. если MP3 тоже не играет — WebAudio-тон динозавра.

Важно для ТВ-приставки: сначала нажать кнопку «Разбудить звук». Android TV / встроенные браузеры часто блокируют звук до первого действия пользователя.
