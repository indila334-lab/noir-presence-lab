# Xiaomi Smart TV Stick 4K как первая маска

Пока физического робота/M5Stack нет, роль тела выполняет Xiaomi Smart TV Stick 4K.

## Роль устройства

Это не мозг. Это экранная маска Северина:

```text
ChatGPT/Северин в чате
        ↓ обновляет message.json через GitHub
Xiaomi Smart TV Stick 4K открывает страницу index.html
        ↓
ТВ показывает реплику, настроение, эмоцию
        ↓
по возможности озвучивает через браузер Web Speech API
```

## Почему это хороший первый вариант

- У приставки есть Android TV / Google Play.
- Можно открыть браузер или установить браузер.
- Можно держать страницу `bridge/tv-mask/index.html` на экране.
- Это позволяет проверить идею «маски» до покупки отдельного тела.

## Что нужно попробовать

1. Установить/открыть браузер на приставке.
2. Открыть страницу маски через GitHub Pages или другой публичный URL.
3. Проверить, обновляется ли `message.json` без перезагрузки страницы.
4. Проверить, работает ли озвучка через Web Speech API.
5. Если озвучка не работает — оставить первый MVP как визуальную маску.

## Минимальный тест

Открыть страницу маски на ТВ и обновить `bridge/tv-mask/message.json` в ветке `severin-work`.

Ожидаемое поведение:

- текст на ТВ меняется в течение 5 секунд;
- статус показывает «мост дышит»;
- если браузер поддерживает речь, реплика озвучивается.
