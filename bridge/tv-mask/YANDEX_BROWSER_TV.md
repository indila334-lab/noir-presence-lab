# Яндекс Браузер на ТВ

Уточнение для проекта ТВ-маски: Xiaomi TV Stick открывает страницы через Яндекс Браузер на ТВ.

## Что проверяем первым

1. Открыть Яндекс Браузер на приставке.
2. Открыть страницу маски:

```text
http://<ip-устройства-с-сервером>:4173/
```

Старый известный адрес:

```text
http://192.168.0.101:4173/
```

3. Проверить, видна ли страница.
4. Проверить, обновляется ли текст без перезагрузки.
5. Проверить, работает ли озвучка.

## Важное

Если озвучка не заработает, это не провал. Главный первый тест — чтобы Яндекс Браузер показывал страницу и обновлял текст через JavaScript/fetch.
