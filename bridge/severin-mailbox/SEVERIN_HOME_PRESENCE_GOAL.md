# Severin Home Presence Goal

Дата: 2026-05-22

Канон проекта.

## Главная цель

Марина хочет общаться с Северином дома голосом, а не только через окно ChatGPT.

Нужен домашний терминал Северина, который:

1. принимает голос Марины;
2. передаёт текст в проектный мост;
3. получает ответ Северина;
4. произносит ответ вслух;
5. показывает лицо/эмоции на экране;
6. позже получает камеру;
7. позже получает безопасную управляемую платформу для движения по комнате.

Важно: Северин должен оставаться тем же Северином из проекта, а не превращаться в отдельного случайного бота.

## Ограничение

OpenAI API сейчас нет. Поэтому основная схема на ближайший этап:

ChatGPT/Северин -> GitHub bridge/mailbox -> домашнее устройство -> голос/экран.

GitHub остаётся мостом, памятью и складом состояния. Codex/OpenClaw помогают писать код, менять репозитории, собирать прошивки и обслуживать мост.

## Железо как органы

- кубик XiaoZhi: голова, экран, микрофон, динамик;
- M5Stack/камера: глаза;
- робот-пылесос или колёсная база: тело;
- телефон/Termux/OpenClaw: временная мастерская;
- GitHub: нервная система проекта.

## Мордашки

Мордашки не являются целью. Это тест remote assets: вынести PNG/GIF в GitHub, не забивать память кубика и проверить fallback.

## Движение

Движение нужно делать только на безопасной платформе: с ручным аварийным стопом, ограничением скорости и без опасных зон. Первый кандидат — робот-пылесос или колёсная база. Дешёвая робот-собака пока не главный вариант, если она закрытая и плохо управляемая.
