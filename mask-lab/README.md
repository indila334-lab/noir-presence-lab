# Severin Mask Lab

Минимальная статическая сцена для проекта маски Северина.

## Что есть

- `index.html` — тёмная сцена и каркас маски.
- `styles.css` — визуальная оболочка, состояния, адаптация под экран ТВ/мобильный браузер.
- `mask.js` — контроллер настроений и простая интерактивность.

## Управление

- `1` — wake / пробуждение
- `2` — focus / фокус
- `3` — grin / улыбка
- `4` — storm / гроза
- `5` — shadow / тень
- `Space` — импульс
- `Enter` или клик/тап — следующее состояние

## Зачем это нужно

Это не финальная маска, а безопасный мост: статический экран, который можно открыть на ТВ-приставке через браузер и постепенно подключать к голосу, камере, эмоциям, API или локальному серверу.
